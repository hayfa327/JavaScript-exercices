# FED27

## Overview

Exercise files for Front-End Development FED27 module at Hyper Island.

## Contents

* 1-10 JavaScript exercises focusing on DOM manipulation, event handling, web APIs, advanced JavaScript exercises on
  asynchronous programming, closures, and design patterns.
* 11-25 React exercises covering component creation, state management, hooks, performance optimization, context API, and
  testing.

## Getting Started

1. Fork the repository:
2. `npm install` to install dependencies.

## Submitting Exercises

1. Complete the exercises in the respective folders.
2. Commit your changes.
3. Push to your forked repository.
4. Submit a pull request for review.
